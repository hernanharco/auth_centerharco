# **App Name**: AuthCenter

## Core Features:

- User Registration: Allows new users to create accounts with username, email, and password.
- User Login: Enables registered users to log in using their credentials and obtain a JWT.
- Profile Management: Allows users to manage their profile information.
- GraphQL API: Provides a GraphQL API for user management and authentication tasks.
- JWT Authentication: Generates and validates JWTs for authenticating requests between services.
- Admin Dashboard: A visual administration panel for managing users, roles, and permissions.
- Email Authentication with Gmail: Allows users to authenticate using their Google account via a Gmail integration, providing a streamlined login experience.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and security.
- Background color: Light gray (#F5F5F5), creating a clean and professional backdrop.
- Accent color: Soft orange (#FFAB40) for interactive elements and key actions.
- Body font: 'Inter', sans-serif, providing a modern and neutral feel. Headline font: 'Space Grotesk', sans-serif. Note: currently only Google Fonts are supported.
- Use clear, minimalist icons from a consistent set to represent different user actions and statuses.
- Design a responsive layout that adapts well to different screen sizes, with a focus on ease of navigation and information clarity.
- Incorporate subtle animations for loading states and transitions to enhance user experience without being distracting.
